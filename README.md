# 📋 Sistema de Cadastro de Pessoas

Sistema simples de cadastro desenvolvido em Python puro.

## 💡 Sobre o projeto

Projeto desenvolvido para praticar os fundamentos de Python:
- Listas e dicionários
- Funções
- Estruturas condicionais (if, elif, else)
- Laços de repetição (while, for)
- Validação de entrada do usuário

## ⚙️ Funcionalidades

- ➕ Cadastrar pessoa (nome, idade, e-mail, telefone)
- 📋 Listar todos os cadastros em tabela
- 🔍 Buscar por nome
- ❌ Remover cadastro por ID

## 🛠️ Tecnologias

- Python 3.x
- Sem dependências externas
